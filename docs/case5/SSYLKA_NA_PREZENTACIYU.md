### Как подготовить файл для ссылки

1. Откройте в VS Code файл [`PREZENTACIYA_KEIS_5_MARP.md`](PREZENTACIYA_KEIS_5_MARP.md).
2. Установите расширение **Marp for VS Code** (или экспортируйте через [Marp CLI](https://github.com/orgs/marp-team/discussions)).
3. Команда экспорта: **Marp: Export Slide Deck** → выберите **PowerPoint (.pptx)** или **PDF**.
4. Загрузите полученный файл в **Google Drive**, **Microsoft OneDrive** или **Яндекс.Диск**.
5. Включите доступ **по ссылке** для просмотра (или только для людей с адресом вуза — по требованиям методички).
6. **Замените** URL выше на вашу реальную ссылку и продублируйте её в тексте отчёта / титульном листе практики.
